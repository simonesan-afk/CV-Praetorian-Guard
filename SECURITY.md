# Security Policy

## Supported Versions

This project is an educational prompt and does not include executable code,
network services, or production deployments.

As such, there are no supported versions in the traditional security sense.
All published versions are provided "as-is" for educational and personal use only.

---

## Reporting a Vulnerability

This project does not process, store, or transmit data on behalf of users.
However, potential issues may include:

- Accidental exposure of personal or sensitive information (PII)
- Misuse of the prompt for hiring, screening, or decision-making purposes
- Ethical or safety concerns related to prompt behavior

If you believe you have identified a security, privacy, or ethical issue:

1. **Do NOT include real personal data** in your report.
2. Open a GitHub Issue describing the concern clearly and concisely.
3. Label the issue as **Security**, **Privacy**, or **Ethical Concern** where applicable.

Reports will be reviewed on a best-effort basis.
There is no guaranteed response time or obligation to provide fixes.

---

## Data Safety Notice

Users are strongly advised to redact all personal data before using this prompt
with any AI system.

This project is not designed to handle real CVs, identifiable personal data,
or confidential information.
